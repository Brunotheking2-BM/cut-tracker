KAKO POSTAVITI SYNC
  1. Idi na github.com/settings/tokens/new (već je predispunjeno)
  2. Pod "Expiration" izaberi "No expiration" (ili 90 dana)
  3. Provjeri da je "gist" checkbox uključen
  4. Klikni "Generate token" na dnu
  5. Kopiraj token (počinje s ghp_)
  6. Vrati se u app → Setup tab → zalijepi token u "GitHub token"
  7. Klikni "SPREMI POSTAVKE"
  8. App će automatski napraviti private Gist i sve syncat
  9. Na drugom uređaju: otvori isti HTML, unesi isti token + isti Gist ID
Sigurnost: Token se sprema samo lokalno u tvoj browser. Gist je private (samo ti ga vidiš). Ima samo "gist" permission, ne može pristupit ničemu drugom na tvom GitHub računu.
